# COLDWANDE CBT Platform

## How to Run
1.  **Open the Application**: Double-click on `index.html` to open it in your default web browser.
2.  **Select Level**: On the homepage, click on **200 Level**.
3.  **Start Exam**: Scroll down to find **Statistical Inference (STT201)** and click **Start Practice**.
4.  **Take the Test**: You will see the questions you added. Select answers and navigate using Next/Prev.

## Managing Questions
- To add more questions for STT201, edit `data/stt201.json`.
- Ensure the format matches the existing questions.
- A **hard refresh** (Ctrl+F5) may be required after changing JSON files to update the LocalStorage.

## Troubleshooting
- If questions don't appear, try clearing your browser's Local Storage (F12 -> Application -> Storage -> Clear Site Data) and reload the page.
